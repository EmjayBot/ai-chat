Admin Setup: Enable CORS for live forum search
▼

To allow this page to fetch search results directly from your Discourse:

  1. Discourse Admin → Settings → Security
  2. Find cors_origins → add your GitHub Pages URL (e.g., https://yourname.github.io)
  3. (Optional) Enable cors enabled if present
  4. Save. No server code needed.

Without CORS, the chat still works—but live search will show "Enable CORS in Discourse admin".
